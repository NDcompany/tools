## What It Checks

### Meta Tag Analysis
- Title tag presence and optimization
- Meta description length and quality
- Viewport and mobile compatibility
- Essential meta tags

### Content Gap Analysis
- Compare keywords with up to 5 competitors
- Identify missing high-value keywords
- Frequency analysis of content terms

### Title Optimization
- Length validation (30-60 characters)
- Power word suggestions
- SEO best practices

### Actionable Recommendations
- Specific, prioritized action items
- Quick wins for immediate improvement
- Technical and content suggestions

## How to Use

1. **Enter Your URL** - Paste your website URL in the main input
2. **Add Competitors** (Optional) - Enter up to 5 competitor URLs
3. **Click Audit** - Wait 30 seconds for comprehensive analysis
4. **Review Results** - Get color-coded issues and recommendations
5. **Export Report** - Download as CSV or HTML for sharing

## Technical Details

- **Frontend**: Vanilla JavaScript, TailwindCSS
- **No Backend Required** - Pure client-side application
- **No Data Collection** - Everything runs in your browser
- **Browser Support**: Modern browsers (Chrome, Firefox, Safari, Edge)

## Local Development

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start auditing websites immediately

No build process, no dependencies, no complexity.

## CORS Limitations

Due to browser security policies, some websites may not be accessible for direct analysis. The tool will show appropriate error messages and suggestions in these cases.
